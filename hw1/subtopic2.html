The management of memory is another crucial part of operating systems. Program memory allocation and deallocation during execution are managed by the operating system. The OS makes effective use of the limited physical memory using strategies including paging, segmentation, and virtual memory. By employing disk space as an extension of RAM, virtual memory in particular enables computers to execute huge programs, giving users the appearance that they have more memory than is actually available.
